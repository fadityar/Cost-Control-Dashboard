# 🍽️ F&B Cost Control Dashboard - Hospitality Management System

Dashboard manajemen biaya makanan dan minuman (F&B Cost Control) untuk industri hospitality yang komprehensif dan mudah digunakan.

## 📋 Deskripsi

**F&B Cost Control Dashboard** adalah platform digital yang dirancang khusus untuk hotel, restoran, dan layanan katering untuk:

- 📊 Memantau biaya makanan dan minuman secara real-time
- 📈 Menganalisis tren biaya dan performa
- 💾 Mengelola inventory dengan efisien
- 👥 Mengelola supplier dan purchase order
- 📄 Membuat laporan dan analisis mendalam
- 🎯 Mencapai target cost control yang optimal

## ✨ Fitur Utama

### 1. **Dashboard Utama**
- KPI Cards untuk metrik penting:
  - Food Cost %
  - Total Cost Bulanan
  - Beverage Cost %
  - Inventory Value
- Chart interaktif dengan trend 6 bulan
- Distribusi biaya per kategori

### 2. **Manajemen Inventory**
- Daftar lengkap item dengan kategori
- Stock level monitoring
- Unit cost tracking
- Status warning (Low Stock, Critical, Normal)
- Edit/update functionality
- Search dan filter

### 3. **Manajemen Supplier**
- Daftar supplier dengan rating
- Informasi kontak dan payment terms
- Minimum order info
- Contact dan edit actions
- Performance tracking

### 4. **Laporan & Analisis**
- Daily Cost Report
- Monthly Analysis
- Variance Report
- Supplier Performance Report
- Download PDF functionality

### 5. **Pengaturan Sistem**
- Konfigurasi target cost
- Currency selection
- User management
- Role-based access (Admin, Manager, Staff)

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Struktur markup
- **CSS3** - Styling responsif
- **JavaScript (Vanilla)** - Interaktivitas
- **Chart.js 3.9.1** - Visualisasi data
- **Font Awesome 6.4.0** - Icon library

## 🚀 Cara Menggunakan

### 1. **Clone/Download Repository**
```bash
git clone https://github.com/fadityar/Cost-Control-Dashboard.git
cd Cost-Control-Dashboard
```

### 2. **Buka di Browser**
- Buka file `index.html` langsung di browser
- Atau gunakan Live Server (VS Code extension)
- Dashboard akan tampil dengan semua fitur

### 3. **Navigasi**
- Click menu untuk berpindah antar section

- Inventory: Manage stock dan items
- Suppliers: Kelola supplier list
